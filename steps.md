# solyshi-workstation — Setup Protocol

Complete installation protocol for the solyshi Arch Linux workstation setup.
Kernel: linux-lts | WM: Hyprland | Shell: Zsh | Terminal: Kitty

---

## 1. Base System

```bash
sudo systemctl enable --now NetworkManager
```

- Base packages: `base base-devel linux linux-lts linux-firmware linux-headers linux-lts-headers`
- Boot: systemd-boot (no GRUB), LTS as default entry in `/boot/loader/loader.conf`
- AUR helper: yay
  ```bash
  git clone https://aur.archlinux.org/yay.git && cd yay && makepkg -si
  yay -Y --gendb && yay -Syu --devel && yay -Y --devel --save
  ```

```bash

```

- Oh-My-Zsh via curl installer
- Plugins: zsh-autosuggestions, zsh-syntax-highlighting (both cloned into ~/.oh-my-zsh/custom)
- Theme: Powerlevel10k
  ```bash
  git clone --depth=1 https://github.com/romkatv/powerlevel10k.git \
    ${ZSH_CUSTOM:-$HOME/.oh-my-zsh/custom}/themes/powerlevel10k
  ```
- Kitty: padding 12px, `placement_strategy center`, matugen colors via `include colors.conf`
- zoxide + eza navigation

## 16. System Stability

### Snapshots (Timeshift)
```bash
sudo pacman -S timeshift
```
- Mode: rsync (no btrfs)
- Target: `/mnt/archive` (UUID: 3f855919-60d7-4682-83b4-27523566ba9a)
- Schedule: weekly, 2 snapshots
- Exclusions: `/mnt/**`, `/tmp/**`, `/var/log/**`, `/var/cache/pacman/pkg/**`, `/home/solyshi/**`

### systemd-boot LTS
```bash
# /boot/loader/loader.conf
default 2026-01-20_20-56-11_linux-lts.conf
timeout 3
```

### fstab — nofail for external disks
Mount all HDDs/SSDs that are not strictly required at boot with `nofail`:
```
UUID=xxxx /mnt/backup ext4 defaults,nofail 0 2
```
Prevents emergency mode when devices are unavailable.

### Btop — Resource Manager
```bash
yay -S Btop
```
- Floating toggle scratchpad via mainMod + Shift + Y

## 17. Hyprland — Scratchpads & Workspace Rules

- Workspace assignments: qutebrowser→WS2, dolphin→WS3, thunderbird→WS9
- Scratchpads: Spotify (`Super+Shift+M`), Vesktop (`Super+Shift+D`), Terminal (`Super+Shift+T`)
- Script: `~/.config/hypr/scripts/toggle-scratch-term.sh` for terminal scratchpad
- Smart gaps enabled: single window fills screen without gaps/border
- Keybinds consolidated in `keybinds/main.conf`
- `windowrules.conf` and `layerrules.conf` included in `hyprland.conf`
- Autostart: kitty→WS1, qutebrowser→WS2, thunderbird→WS9 (silent)
- Screenshot keybinds: Print (fullscreen), Shift+Print (region), Ctrl+Print (clipboard)
- Screenshots saved to `~/Pictures/Screenshots/`
- Workspace layout: WS1 kitty, WS2 qutebrowser, WS3 dolphin (manual), WS9 thunderbird, WS10 tradingview (manual)
- TradingView windowrule: workspace 10 (Super+0)

## 18. Tmux — Sessionizer & Workflow

- tpm installed to `~/.config/tmux/plugins/tpm` (not in repo, via .gitignore)
- Plugins: tpm, tmux-sensible, tmux-resurrect, tmux-continuum
- continuum-restore enabled — automatically restores sessions after reboot
- resurrect-capture-pane-contents enabled — saves pane contents
- Keybinds: h/v for splits, r for reload, c for new window in current path

### tmux-sessionizer
Dynamic project management script — searches `~/solyshi-workstation` and `~/projects` for directories, displays them via fzf with live preview.

- Existing session → `switch-client`
- New session → `new-session`
- Path: `~/solyshi-workstation/dotfiles/scripts/scripts/tmux-sessionizer` (linked via stow to `~/scripts/`)

### Hotkey Integration (3 levels)
- **Zsh** `Ctrl+F` — sessionizer directly from the shell
- **Tmux** `Prefix+F` — sessionizer as a centered popup (switch while programs are running)

### Roadmap (not yet implemented)

- [ ] **Harpoon** — quick jumping between project files via `Alt+1/2/3/4`
